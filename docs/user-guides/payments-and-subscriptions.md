# Payments and Subscriptions Guide

## For Members

## Upgrade Your Plan

1. Open `Settings -> Subscription`.
2. Choose a target plan (`Pro` or `Enterprise`).
3. Select a payment method:
   - Card
   - M-PESA
   - Flutterwave
   - Paystack
   - PayPal
4. Click `Choose <Plan>`.
5. Complete payment and return to the confirmation page.

## Confirm Payment

- The app redirects to `/payments/confirmation`.
- If successful, your subscription is updated automatically.
- If payment fails, retry from the subscription page.

## Cancel Subscription

1. Open `Settings -> Subscription`.
2. Click `Cancel Plan`.
3. Confirm status changed to `cancelled`.

## View Transaction History

- Recent transactions appear on the subscription page with:
  - Reference
  - Amount
  - Payment method
  - Status
  - Date
